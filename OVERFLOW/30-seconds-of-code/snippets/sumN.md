Sums all the numbers between `1` and `n`.

- Use the formula `(n * (n + 1)) / 2` to get the sum of all the numbers between 1 and `n`.

```js
const sumN = (n) => (n * (n + 1)) / 2;
```

```js
sumN(100); // 5050
```
