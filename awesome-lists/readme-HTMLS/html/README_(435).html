Awesome Funny Markov Chains

A facetiously curated list of delightfully amusing Markov chain output, inspired by memorylessness.

If you want to contribute to this list (please do), send a pull request or contact me [@sublimino](https://twitter.com/sublimino).

Looking for more information on Markov Chains? Try Awesome Machine Learning.
