# --description--

對網格容器使用 `align-items` 屬性可以讓網格中所有的網格項沿豎直方向對齊。

# --instructions--

請使用這個屬性將所有網格項移動到單元格的末尾。

# --hints--

class 爲 `container` 的元素應具有 `align-items` 屬性且屬性值應爲 `end`。

```js
assert(
  code.match(/.container\s*?{[\s\S]*align-items\s*?:\s*?end\s*?;[\s\S]*}/gi)
);
```
