# --description--

有時你想爲網站添加一個 `a` 元素，但還不確定要將它鏈接到哪裏。

當你使用 `JavaScript` 更改鏈接的指向時，這也很方便，我們將在後面的課程中介紹。

# --instructions--

`href` 屬性的當前值是指向 “`https://www.freecatphotoapp.com`”。 請將 `href` 屬性的值替換爲`#`，以此來創建鏈接佔位符。

例如: `href="#"`

# --hints--

`a` 的 `href` 屬性值應爲 "#"。

```js
assert($('a').attr('href') === '#');
```
