# --description--

通過 `font-family` 屬性，我們可以設置元素裏的字體族名。

如果你想將 `h2` 元素的字體設置爲 `sans-serif`，可以這樣寫：

```css
h2 {
  font-family: sans-serif;
}
```

# --instructions--

請將 `p` 元素的字體設置爲 `monospace`。

# --hints--

`p` 元素應使用 `monospace` 作爲字體。

```js
assert(
  $('p')
    .not('.red-text')
    .css('font-family')
    .match(/monospace/i)
);
```
