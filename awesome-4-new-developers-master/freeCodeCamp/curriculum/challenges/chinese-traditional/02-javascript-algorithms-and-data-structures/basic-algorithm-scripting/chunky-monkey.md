# --description--

請編寫一個函數，該函數將一個數組（第一個參數）拆分成若干長度爲 `size`（第二個參數）的子數組，並將它們作爲二維數組返回。

# --solutions--

```js
function chunkArrayInGroups(arr, size) {
  let out = [];

  for (let i = 0; i < arr.length; i += size) {
    out.push(arr.slice(i, i + size));
  }

  return out;
}

chunkArrayInGroups(["a", "b", "c", "d"], 2);
```
