# --description--

一個將數據添加到數組末尾的簡單方法是 `push()` 函數。

`.push()` 接受一個或多個參數（<dfn>parameters</dfn>），並把它壓入到數組的末尾。

示例：

```js
var arr1 = [1,2,3];
arr1.push(4);

var arr2 = ["Stimpson", "J", "cat"];
arr2.push(["happy", "joy"]);
```

`arr1` 現在值爲 `[1, 2, 3, 4]`，`arr2` 值爲 `["Stimpson", "J", "cat", ["happy", "joy"]]`。
