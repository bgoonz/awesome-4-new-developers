# --description--

在 JavaScript 中，你也可以用小數進行計算，就像整數一樣。

把兩個小數相乘，並得到它們乘積。

# --instructions--

改變 `0.0` 的數值讓變量 product 的值等於 `5.0`。

# --hints--

變量 `product` 應該等於 `5.0`。

```js
assert(product === 5.0);
```

要使用 `*` 運算符。

```js
assert(/\*/.test(code));
```
