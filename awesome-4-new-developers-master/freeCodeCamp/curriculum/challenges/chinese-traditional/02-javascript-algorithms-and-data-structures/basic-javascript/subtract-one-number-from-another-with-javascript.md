# --description--

我們也可以在 JavaScript 中進行減法運算。

JavaScript 中使用 `-` 來做減法運算。

**示例**

```js
myVar = 12 - 6;
```

現在，變量 `myVar` 的值爲 `6`。
# --instructions--

改變數字`0`讓變量 difference 的值爲 `12`。
