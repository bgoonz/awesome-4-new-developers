# --description--

與 `+=` 操作符類似，`-=` 操作符用來對一個變量進行減法賦值操作。

```js
myVar = myVar - 5;
```

將從 `myVar` 中減去 `5`。 等價於：

```js
myVar -= 5;
```

# --instructions--

使用 `-=` 操作符對 `a`、`b` 和 `c` 實現相減賦值操作。

# --hints--

```js
assert(a === 5);
```

`b` 應該等於 `-6`。

```js

```

`c` 應該等於 `2`。

