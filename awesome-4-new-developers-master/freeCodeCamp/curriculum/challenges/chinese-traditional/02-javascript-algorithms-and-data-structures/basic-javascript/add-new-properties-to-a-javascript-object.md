# --description--

你也可以像更改屬性一樣給 JavaScript 對象添加屬性。

這裏展示瞭如何給 `ourDog` 添加一個屬性 `bark`：

```js
ourDog.bark = "bow-wow";
```

或者

```js
ourDog["bark"] = "bow-wow";
```

現在，當我們執行 `ourDog.bark` 時，就能得到他的叫聲，`bow-wow`。

例如：

```js
var ourDog = {
  "name": "Camper",
  "legs": 4,
  "tails": 1,
  "friends": ["everything!"]
};

ourDog.bark = "bow-wow";
```
