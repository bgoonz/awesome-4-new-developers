# --description--

`some` 方法用於檢測數組中*任何*元素是否滿足指定條件。 如果有一個元素滿足條件，返回布爾值 `true`，反之返回 `false`。

舉個例子，下面的代碼檢測數組`numbers`中是否有元素小於 10：

```js
var numbers = [10, 50, 8, 220, 110, 11];
numbers.some(function(currentValue) {
  return currentValue < 10;
});
```

`some` 方法將返回 `true`。
