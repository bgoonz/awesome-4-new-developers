## --text--

您將使用什麼 SQL 命令來檢索具有電子郵件地址 `quincy@freecodecamp.org` 的所有用戶？

## --answers--

```sql
SELECT Users WHERE email="quincy@freecodecamp.org"
```

---

```sql

```

---


---

```sql
SELECT * FROM Users WHERE email IS "quincy@freecodecamp.org"
```

