# --description--

Como o operador `+=`, `-=` subtrai um número de uma variável.

```js
myVar = myVar - 5;
```

vai subtrair `5` de `myVar`. Essa expressão pode ser reescrita assim:

```js
myVar -= 5;
```

# --instructions--

Converta as atribuições para `a`, `b` e `c` para usar o operador `+=`.

