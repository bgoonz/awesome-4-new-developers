# --description--

使用自减符号 `--`，你可以很方便地对一个变量执行<dfn>自减</dfn>或者 -1 运算。

```js
i--;
```

等效于：

```js
i = i - 1;
```

**注意：**`i--;` 这种写法省去了书写等号的必要。
