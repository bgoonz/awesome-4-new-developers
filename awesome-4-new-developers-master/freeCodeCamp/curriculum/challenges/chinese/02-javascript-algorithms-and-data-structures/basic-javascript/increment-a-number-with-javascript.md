# --description--

使用 `++`，我们可以很容易地对变量进行<dfn>自增</dfn>或者 +1 运算。

```js
i++;
```

等效于：

```js
i = i + 1;
```

**注意：**`i++;` 这种写法省去了书写等号的必要。
