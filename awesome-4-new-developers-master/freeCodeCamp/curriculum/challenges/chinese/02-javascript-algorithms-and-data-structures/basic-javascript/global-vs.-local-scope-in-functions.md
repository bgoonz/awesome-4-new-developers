# --description--

一个程序中有可能具有相同名称的<dfn>局部</dfn>变量 和<dfn>全局</dfn>变量。 在这种情况下，局部变量将会优先于全局变量。

下面为例：

```js
var someVar = "Hat";
function myFun() {
  var someVar = "Head";
  return someVar;
}
```

函数 `myFun` 将会返回字符串 `Head`，因为局部变量的优先级更高。
