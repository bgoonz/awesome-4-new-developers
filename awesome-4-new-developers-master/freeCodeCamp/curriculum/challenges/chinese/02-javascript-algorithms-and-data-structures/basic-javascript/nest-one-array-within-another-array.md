# --description--

您也可以在其他数组中嵌套数组，如：

```js
[["Bulls", 23], ["White Sox", 45]]
```

这也叫做多维数组（<dfn>multi-dimensional array</dfn>）。

# --instructions--

创建一个名为 `myArray` 的嵌套数组。

# --hints--

`myArray` 应该至少有一个数组嵌套在另一个数组中。

```js
assert(Array.isArray(myArray) && myArray.some(Array.isArray));
```
