# --description--

严格不相等运算符（`!==`）与全等运算符是相反的。 这意味着严格不相等并返回 `false` 的地方，用严格相等运算符会返回 `true`，*反之亦然*。 严格不相等运算符不会转换值的数据类型。

**示例**

```js
3 !==  3
3 !== '3'
4 !==  3
```

按顺序，这些表达式会返回 `false`、`true`、`true`。

# --instructions--

在 `if` 语句中，添加严格不相等运算符，这样函数在当 `val` 不严格等于 `17` 的时候，会返回 `Not Equal`。
