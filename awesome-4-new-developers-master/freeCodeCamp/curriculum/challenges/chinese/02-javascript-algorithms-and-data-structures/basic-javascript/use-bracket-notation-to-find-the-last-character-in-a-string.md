# --description--

要获取字符串的最后一个字符，可以用字符串的长度减 1 的索引值。

例如，如果 `var firstName = "Ada"` 中，那么你可以通过 `firstName[firstName.length - 1]` 来得到字符串的最后的一个字符。

示例：

```js
var firstName = "Ada";
var lastLetter = firstName[firstName.length - 1];
```

`lastLetter` 值为字符串 `a`。
