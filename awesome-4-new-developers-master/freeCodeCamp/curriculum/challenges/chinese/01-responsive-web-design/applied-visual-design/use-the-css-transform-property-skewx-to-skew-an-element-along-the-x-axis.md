# --description--

接下来要介绍的 `transform` 属性是 `skewX()`：它使选择的元素沿着 X 轴（横向）翻转指定的角度。

下面的代码沿着 X 轴翻转段落元素 -32 度。

```css
p {
  transform: skewX(-32deg);
}
```

# --instructions--

使用 `transform` 属性沿 X 轴翻转 id 为 `bottom` 的元素 24 度。
