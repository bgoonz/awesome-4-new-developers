# --instructions--

在 `tests/1_unit-tests.js` 中，`Strings` 套件里标有 `#13` 的测试下，将每个 `assert` 改成 `assert.isString` 或 `assert.isNotString` 方法，通过测试（结果应该返回 `true`）。 不要修改传给断言的参数。
