# --description--

A differenza delle stringhe, gli elementi degli array sono <dfn>mutabili</dfn> e possono essere cambiati liberamente.

**Esempio**

```js
var ourArray = [50,40,30];
ourArray[0] = 15;
```

`ourArray` ha ora il valore `[15, 40, 30]`.

**Note:** Non ci dovrebbero essere spazi tra il nome dell'array e le parentesi quadre, come in `array [0]`. Anche se JavaScript Ã¨ in grado di elaborarlo correttamente, questo potrebbe confondere altri programmatori che leggessero il tuo codice.
