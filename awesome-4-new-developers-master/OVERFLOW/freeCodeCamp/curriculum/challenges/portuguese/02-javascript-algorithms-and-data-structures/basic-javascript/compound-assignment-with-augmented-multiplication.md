# --description--

O operador `*=` multiplica uma variável por um número.

```js
myVar = myVar * 5;
```

multiplicará `myVar` por `5`. Essa expressão pode ser reescrita assim:

```js
myVar *= 5;
```

# --instructions--

Converta as atribuições para `a`, `b` e `c` para usar o operador `*=`.

# --hints--

```js
assert(a === 25);
```


```js
assert(c === 46);
```

```js
assert(code.match(/\*=/g).length === 3);
```

```js
assert(

    /var c = 4\.6;/.test(code)
);
```
