# --description--

Escreva uma função que retorne o fatorial de um número.

O fatorial de um número é dado por:

<pre><big>n! = n * (n-1) * (n-2) * ..... * 1</big>
</pre>

Por exemplo:

<ul>
  <li><code>3! = 3 * 2 * 1 = 6</code></li>
  <li><code>4! = 4 * 3 * 2 * 1 = 24</code></li>
</ul>

**Observação:** `0! = 1`

# --solutions--

```js
function factorial(n) {
  let sum = 1;
  while (n > 1) {
    sum *= n;
    n--;
  }
  return sum;
}
```
