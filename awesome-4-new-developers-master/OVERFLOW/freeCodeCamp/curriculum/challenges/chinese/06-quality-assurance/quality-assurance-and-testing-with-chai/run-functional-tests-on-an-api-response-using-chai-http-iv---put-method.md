# --instructions--

在 `tests/2_functional-tests.js` 中，修改 `'send {surname: "da Verrazzano"}'` 测试（`// #4`）。

发送以下 JSON 响应作为有效载荷到 `/travellers` 路径：

```json
{
  "surname": "da Verrazzano"
}
```

在 `request.end` 返回中检查以下情况：

1.  `status`
2.  `type`
3.  `body.name`
4.  `body.surname`

请按照以上顺序书写断言，顺序错误会影响系统判定。 完成后请务必删除 `assert.fail()`。
