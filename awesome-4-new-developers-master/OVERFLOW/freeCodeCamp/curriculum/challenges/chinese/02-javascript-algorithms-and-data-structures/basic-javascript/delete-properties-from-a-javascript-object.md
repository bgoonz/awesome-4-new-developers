# --description--

我们同样可以删除对象的属性，例如：

```js
delete ourDog.bark;
```

例如：

```js
var ourDog = {
  "name": "Camper",
  "legs": 4,
  "tails": 1,
  "friends": ["everything!"],
  "bark": "bow-wow"
};

delete ourDog.bark;
```

在上面代码的最后一行中，`ourDog` 是这样的：

```js
{
  "name": "Camper",
  "legs": 4,
  "tails": 1,
  "friends": ["everything!"]
}
```

# --instructions--

删除 `myDog` 对象的 `tails` 属性。 可以使用点操作符或者中括号操作符。
