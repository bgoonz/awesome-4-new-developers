# --description--

`*=` 操作符是让变量与一个数相乘并赋值。

```js
myVar = myVar * 5;
```

将 `myVar` 乘以 `5`。 等价于：

```js
myVar *= 5;
```

# --instructions--

使用 `*=` 操作符对 `a`、`b` 和 `c` 实现赋值相乘操作。
