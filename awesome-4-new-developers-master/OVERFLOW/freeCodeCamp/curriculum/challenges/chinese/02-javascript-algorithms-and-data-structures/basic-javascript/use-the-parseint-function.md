# --description--

`parseInt()` 函数解析一个字符串返回一个整数。 下面是一个示例：

```js
var a = parseInt("007");
```

上述函数将字符串 `007` 转换为整数 `7`。 如果字符串中的第一个字符不能转换为数字，则返回 `NaN`。
