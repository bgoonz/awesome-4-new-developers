# --description--

与字符串的数据不可变不同，数组的数据是可变的（ <dfn>mutable</dfn>），可以自由地改变。

**示例**

```js
var ourArray = [50,40,30];
ourArray[0] = 15;
```

`ourArray` 值为 `[15, 40, 30]`。

**注意：** 数组名与方括号之间不应该有任何空格，比如 `array [0]` 。 尽管 JavaScript 能够正确处理这种情况，但是当其他程序员阅读你写的代码时，这可能让他们感到困惑。

# --instructions--

将数组 `myArray` 中索引为 `0` 上的值修改为 `45`。
