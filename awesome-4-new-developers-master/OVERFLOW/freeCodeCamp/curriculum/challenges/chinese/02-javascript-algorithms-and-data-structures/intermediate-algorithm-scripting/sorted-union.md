# --description--

编写一个带有两个或更多数组的函数，并按原始提供的数组的顺序返回一个新的唯一值数组。

换句话说，所有数组中出现的所有值都应按其原始顺序包括在内，但最终数组中不得重复。

去重后的数字应按其出现在参数中的原始顺序排序，最终数组不应按数字大小进行排序。

如有疑问，请先浏览下方的测试用例。

assert.deepEqual(uniteUnique([1, 3, 2], [5, 2, 1, 4], [2, 1]), [1, 3, 2, 5, 4]);
