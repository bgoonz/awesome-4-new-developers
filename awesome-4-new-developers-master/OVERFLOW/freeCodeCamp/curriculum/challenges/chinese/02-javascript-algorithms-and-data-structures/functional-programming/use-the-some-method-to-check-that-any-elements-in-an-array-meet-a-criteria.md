# --description--

`some` 方法用于检测数组中*任何*元素是否满足指定条件。 如果有一个元素满足条件，返回布尔值 `true`，反之返回 `false`。

举个例子，下面的代码检测数组`numbers`中是否有元素小于 10：

```js
var numbers = [10, 50, 8, 220, 110, 11];
numbers.some(function(currentValue) {
  return currentValue < 10;
});
```

`some` 方法将返回 `true`。
