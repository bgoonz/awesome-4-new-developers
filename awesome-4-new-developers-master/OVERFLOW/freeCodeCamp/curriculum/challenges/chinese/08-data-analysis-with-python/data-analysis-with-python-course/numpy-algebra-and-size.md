## --text--

内存中，对象的大小（例如列表和数据类型）在 Python 标准库和 NumPy 库之间有什么关系？ 知道这一点，对性能有何影响？

## --answers--

标准的 Python 对象占用了比 NumPy 对象更多的内存；标准的 Python 和 NumPy 对象完成的操作时间是大致相同的。

---

Numpy 对象比标准的 Python 对象占用更多的内存；Numpy 的对象相比较标准的 Python 更快地完成操作。
