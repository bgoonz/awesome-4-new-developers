# --description--

现在让我们来修改文本的颜色。

我们通过修改 `h2` 元素的 `style` 属性来改变文本颜色。

我们需要修改 `color` 属性值。

以下是将 `h2` 元素设置为蓝色的方法：

```html
<h2 style="color: blue;">CatPhotoApp</h2>
```

请注意，需要在内联 `style` 声明末尾加上 `;`。

# --instructions--

修改 `h2` 的样式，使它的文本颜色为红色。

# --hints--

`h2` 元素应有一个 `style` 声明。

```js
assert($('h2').attr('style'));
```

`h2` 元素的颜色应设置为 `red`。

```js
assert($('h2')[0].style.color === 'red');
```

`style` 声明应该以 `;` 结尾。

```js
assert($('h2').attr('style') && $('h2').attr('style').endsWith(';'));
```
