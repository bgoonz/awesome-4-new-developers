# --description--

现在让我们来创建一个 Web 表单。

`input` 输入框可以让你轻松获得用户的输入。

你可以像这样创建一个文本输入框：

```html
<input type="text">
```

注意 `input` 输入框是没有结束标签的。

# --hints--

网页中应存在一个类型为 `text` 的 `input` 输入框。

```js
assert($('input[type=text]').length > 0);
```
