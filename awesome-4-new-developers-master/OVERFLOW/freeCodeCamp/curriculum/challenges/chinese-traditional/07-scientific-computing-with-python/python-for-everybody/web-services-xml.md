# --question--

## --text--

以下 XML 有什麼問題？

```xml
<person>
  <name>Chuck</name>
  <phone type="intl">
    +1 734 303 4456
  <email hide="yes" />
</person>
```

## --answers--

Email 標籤缺少結束標籤。

---

間距將導致 XML 無效。

---

Phone 標籤缺少結束標籤。

---

純文本應使用 UTF-8 編碼。
