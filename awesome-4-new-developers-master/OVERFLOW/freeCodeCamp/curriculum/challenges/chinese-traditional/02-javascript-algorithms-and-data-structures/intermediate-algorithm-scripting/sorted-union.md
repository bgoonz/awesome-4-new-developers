# --description--

編寫一個帶有兩個或更多數組的函數，並按原始提供的數組的順序返回一個新的唯一值數組。

換句話說，所有數組中出現的所有值都應按其原始順序包括在內，但最終數組中不得重複。

去重後的數字應按其出現在參數中的原始順序排序，最終數組不應按數字大小進行排序。

如有疑問，請先瀏覽下方的測試用例。
