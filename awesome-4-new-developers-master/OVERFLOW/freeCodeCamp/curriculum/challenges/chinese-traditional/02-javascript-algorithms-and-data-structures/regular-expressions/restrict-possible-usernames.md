# --description--

用戶名在互聯網上隨處可見。 它們是用戶在自己喜歡的網站上的唯一身份。

需要檢索數據庫中的所有用戶名。 以下是用戶在創建用戶名時必須遵守的一些簡單規則。

1) 用戶名只能是數字字母字符。

2) 用戶名中的數字必須在最後。 數字可以有零個或多個。 用戶名不能以數字開頭。

3) 用戶名字母可以是小寫字母和大寫字母。

4) 用戶名長度必須至少爲兩個字符。 兩位用戶名只能使用字母。
