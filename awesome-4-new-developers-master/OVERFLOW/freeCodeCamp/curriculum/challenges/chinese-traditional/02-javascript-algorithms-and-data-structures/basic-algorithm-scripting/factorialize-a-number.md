# --description--

返回一個給定整數的階乘計算結果。

對於整數 `n`，n 的階乘就是所有小於等於 `n` 的正整數的乘積。

階乘通常用符號 `n!` 來表示。

例如：`5! = 1 * 2 * 3 * 4 * 5 = 120`

在這個挑戰中，只有非負整數會作爲參數傳入函數。

```js
function factorialize(num) {
  return num < 1 ? 1 : num * factorialize(num - 1);
}

factorialize(5);
```
