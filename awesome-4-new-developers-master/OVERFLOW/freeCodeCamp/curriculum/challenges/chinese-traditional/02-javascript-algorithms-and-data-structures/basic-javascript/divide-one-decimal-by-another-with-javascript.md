# --description--

現在讓我們將一個小數除以另一個小數。

# --instructions--

改變數值 `0.0` 的值讓變量 `quotient` 的值等於 `2.2`。

# --hints--

`quotient` 的值應該等於`2.2`。

```js
assert(quotient === 2.2);
```

使用 `/` 運算符將 4.4 除以 2。

```js
assert(/4\.40*\s*\/\s*2\.*0*/.test(code));
```

quotient 變量應該只被賦值一次。

```js
assert(code.match(/quotient/g).length === 1);
```
