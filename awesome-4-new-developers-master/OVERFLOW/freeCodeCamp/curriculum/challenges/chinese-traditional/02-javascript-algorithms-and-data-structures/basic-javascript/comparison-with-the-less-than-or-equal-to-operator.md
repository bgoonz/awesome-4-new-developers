# --description--

使用小於等於運算符（`<=`）比較兩個數字的大小。 如果在小於等於運算符左邊的數字小於或者等於右邊的數字，它會返回 `true`。 如果在小於等於運算符左邊的數字大於右邊的數字，它會返回 `false`。 與相等運算符類似，小於或等於運算符會轉換數據類型。

**例如**

```js
4   <= 5
'7' <= 7
5   <= 5
3   <= 2
'8' <= 4
```

按順序，這些表達式會返回 `true`、`true`、`true`、`false` 和 `false`。
