# --description--

使用 `++`，我們可以很容易地對變量進行<dfn>自增</dfn>或者 +1 運算。

```js
i++;
```

等效於：

```js
i = i + 1;
```

**注意：**`i++;` 這種寫法省去了書寫等號的必要。

# --instructions--

修改代碼，使用 `++` 來對變量 `myVar` 進行自增操作。
