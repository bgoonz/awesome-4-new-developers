# --description--

接下來要介紹的 `transform` 屬性是 `skewX()`：它使選擇的元素沿着 X 軸（橫向）翻轉指定的角度。

下面的代碼沿着 X 軸翻轉段落元素 -32 度。

```css
p {
  transform: skewX(-32deg);
}
```

# --instructions--

使用 `transform` 屬性沿 X 軸翻轉 id 爲 `bottom` 的元素 24 度。
