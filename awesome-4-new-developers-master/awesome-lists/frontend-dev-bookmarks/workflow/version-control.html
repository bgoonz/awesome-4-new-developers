Version Control

Context: frontend-dev-bookmarks / Workflow

Version control or source control is a system that records changes to a file or set of files over time so that you can recall specific versions later.

+ Git: Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. + Become a Git Guru: A series of Git tutorials by Atlassian. + OctoLinker: The OctoLinker is a browser extensions which makes references to other files in GitHub clickable.
