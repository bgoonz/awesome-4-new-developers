Web Animations API

Context: frontend-dev-bookmarks / Animation

Web Animations is a new JavaScript API for driving animated content on the web. By unifying the animation features of SVG and CSS, Web Animations unlocks features previously only usable declaratively, and exposes powerful, high-performance animation capabilities to developers.

+ Are we animated yet?: This page tracks the progress of implementing the Web Animations API in Firefox. + WAAPI Browser Support Test (+ Polyfill): This codepen tests whether and to which extend your browser supports Web Animations API. The test is run after including the Polyfill. + Web Animations Polyfill: JavaScript implementation of the Web Animations API.
