Awesome Pinned Gists

A curated list of open source tools for adding dynamic data to pinned gists.

Contents

  • External Services
  • GitHub
  • Resources

External Services

Displaying data from external services in a pinned gist.

  • bird-box - Update a pinned gist to contain the latest tweets of a Twitter user.
  • waka-box - Update a pinned gist to contain your weekly WakaTime stats.
  • waka-box-go - A Golang implementation of waka-box. Update a pinned gist to contain your weekly WakaTime stats.
  • strava-box - Update a pinned gist to contain your YTD exercise metrics from Strava.
  • music-box - Update a pinned gist to contain your weekly listening report on Last.fm.
  • spotify-box - Update a pinned gist to contain your weekly top tracks on Spotify.
  • netease-music-box - Update a pinned gist to contain your weekly listening report on Netease Cloud Music.
  • bilibili-box - Update a pinned gist to contain your latest Bilibili videos.
  • rescue-box - Update a pinned gist to contain your daily productivity stats from RescueTime.
  • neko-box - Update a pinned gist to contain the latest activity from AniList.
  • douban-box - Update a pinned gist to contain the latest user status about books/movies/music from Douban.
  • movie-box - Update a pinned gist to contain your media center stats from Trakt.tv.
  • covid-box - Update a gist to contain global or country specific coronavirus stats.
  • shortcut-box - Update a pinned gist to contain random IDE shortcut
  • youtube-box - Update a pinned gist to show your latest YouTube videos
  • book-box - Update a pinned gist to contain your latest reads from goodreads
  • goodreads-box - Update a pinned gist to show your currently reading books and progress on Goodreads.
  • toggl-box - Update a pinned gist to contain your weekly Toggl time tracking stats
  • steam-box - Update a pinned gist to contain your Steam playtime leaderboard.
  • codestats-box - Update a pinned gist to contain your daily Code::Stats.
  • codestats-box-python - A Python implementation of codestats-box. Update a pinned gist to contain your Code::Stats stats.
  • typeracer-box - Update a pinned gist to contain your latest TypeRacer races
  • chess-com-box-py - Update a pinned gist to contain your Chess.com Ratings.
  • blog-box - Update a pinned gist to show your latest dev.to blog post.
  • medium-stat-box - Update a pinned gist to show your medium stats and latest articles.
  • hitokoto-box - Update a pinned gist to contain a random hitokoto.

GitHub

Displaying more detailed GitHub user data in a pinned gist.

  • activity-box - Update a pinned gist to contain the latest activity of a GitHub user.
  • productive-box - Update a pinned gist to contain your most productive hours during the day.
  • github-stats-box - Update a pinned gist to contain your GitHub statistics.
  • lang-box - Update a pinned gist to contain languages of your recent commits in GitHub

Resources

Resources for keeping pinned gists updated.

  • gist-box - A helper class for updating single-file Gists.
