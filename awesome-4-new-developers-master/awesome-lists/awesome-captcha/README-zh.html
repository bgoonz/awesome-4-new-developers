Awesome Captcha

CAPTCHA is a type of challenge–response test used in computing to determine whether or not the user is human.

Libraries

  • Captcha - Captcha for Laravel 5.

  • Captcha - A Lightweight Pure JavaScript Captcha for Node.js. No C/C++, No ImageMagick, No Canvas.
  • Patchca - Simple yet powerful CAPTCHA library written in Java.

Generation

  • Captcha - Go package captcha implements generation and verification of image and audio CAPTCHAs.
  • Captcha - A captcha library that generates audio and image CAPTCHAs.
  • SVG-captcha - 用node生成svg格式的验证码.
  • Ccap - Node.js generate captcha using c++ library CImg without install any other lib or software.
  • Captchagen - Captcha generation for Node.js.
  • CaptchaImageView - 自定义ImageView生成验证码图像。
  • SwipeCaptcha - Android平台的滑动验证码。
  • Base64Captcha - 快速生成base64编码图片验证码字符串.base64图形验证码(captcha)为golang而设计.
